# AK OS v8.2 — Alta y aprobación de distribuidores

1. Ejecuta `supabase/ak_os_v8_2_distribuidores.sql` en el SQL Editor del Supabase compartido.
2. Copia el contenido del ZIP de Core encima de `autokeys-lab`.
3. Confirma en Vercel Core que existe `SUPABASE_SERVICE_ROLE_KEY`.
4. Commit + push.
5. Abre `/ak-cloud/distribuidores`.

La aprobación activa el usuario existente creado durante el registro, vincula/crea el cliente de Core, asigna plan y créditos de bienvenida.
