# AK Cloud v3.8 — Créditos y precios

## Instalación
1. Ejecuta en Supabase:
   `supabase/autokeys_file_service_v3.8_creditos.sql`
2. Copia el contenido del ZIP encima del repo `autokeys-file-service`.
3. Commit + push.

## Incluye
- Nueva pantalla `/creditos`
- Saldo de créditos
- Movimientos de recarga/consumo
- Tabla visual de precios por servicio
- Link en el menú lateral
- Campos opcionales de coste en pedidos
