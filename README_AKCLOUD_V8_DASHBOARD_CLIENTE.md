# AK Cloud v8.0 — Dashboard Cliente Premium

## Instalación
1. Copia el contenido del ZIP encima del repositorio `autokeys-file-service`.
2. Confirma que se sustituye `app/dashboard/page.tsx`.
3. Haz commit + push.
4. Vercel desplegará automáticamente.

## Incluye
- Dashboard real para distribuidores.
- Pedidos, saldo, descargas y notificaciones desde Supabase.
- Planes y catálogo de servicios leídos desde la configuración de Core.
- Accesos rápidos, últimos pedidos, archivos listos y estado del laboratorio.
- Diseño racing coherente con el resto de AK Cloud.

No requiere SQL nuevo.
