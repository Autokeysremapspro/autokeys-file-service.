# AK Cloud v3.7 — Notificaciones en tiempo real

## Instalación

1. Ejecuta en Supabase:

```text
supabase/autokeys_file_service_v3.7_notificaciones.sql
```

2. Copia el contenido del ZIP encima del repo `autokeys-file-service`.
3. Commit + push.

## Incluye

- Tabla `file_service_notificaciones`
- Centro `/notificaciones`
- Campana premium en dashboard
- Marcar leído / leer todo / eliminar
- Trigger automático cuando cambia el estado de un pedido
- Trigger automático cuando se sube MOD
- Preparado con Supabase Realtime
