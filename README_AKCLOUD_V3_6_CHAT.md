# AK Cloud v3.6 — Chat Técnico

## Instalación

1. Ejecuta en Supabase:

```text
supabase/autokeys_file_service_v3.6_chat.sql
```

2. Copia los archivos encima del repositorio `autokeys-file-service`.
3. Commit + push.

## Incluye

- Tabla `file_service_mensajes`.
- Componente `AKChat`.
- Chat en ficha de pedido del distribuidor.
- Chat en ficha admin del pedido.
- Mensajes en tiempo real mediante Supabase Realtime si está activo.
