# AK Cloud v5.2 — Biblioteca Premium

## Qué incluye

- `/biblioteca` rediseñada como bóveda técnica premium.
- Búsqueda por ORI, MOD, ECU, HW, SW, vehículo, servicio o número FS.
- Filtros ORI / MOD / Todos.
- Cards premium para descargar archivos.
- `/biblioteca/[id]` con ficha del trabajo y todos sus archivos.
- Usa los pedidos existentes de `file_service_pedidos`.
- No requiere SQL nuevo.

## Instalación

Copia encima del repo `autokeys-file-service` y haz:

```bash
git add .
git commit -m "AK Cloud v5.2 Biblioteca Premium"
git push
```
